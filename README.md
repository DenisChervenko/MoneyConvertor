# Currency Quick Convert 🪙

A lightweight Google Chrome extension that allows you to instantly convert currencies by simply selecting text on any webpage. It features a smart overlay button for quick access and full support for multiple exchange rate sources.

## Features

- Smart Detection: Automatically recognizes numbers and currency symbols ($, €, ₽, £, ¥) in the selected text.
- Quick-Access Button: A small icon appears right next to your selection—no need to right-click and open the context menu.
- Multiple Rate Sources: Swap between Central Bank of Russia (CBR), Open ER API, and Currency-API on the fly.
- Rate Pinning: Save your favorite exchange rate provider as the default.

## How to Install (Developer Mode)

Since this extension is hosted here on GitHub, you can easily install it locally in just a few steps:

### 1. Download the Source Code
Clone this repository to your computer or download it as a ZIP archive and unpack it. Make sure to keep the folder in a safe place, as Chrome will read the extension directly from there.

### 2. Open Chrome Extensions Page
Open Google Chrome and navigate to the Extensions management page:
- Either type chrome://extensions/ in the address bar and press Enter.
- Or click the Menu (three dots) -> Extensions -> Manage Extensions.

### 3. Enable Developer Mode
In the top-right corner of the Extensions page, toggle the "Developer mode" switch to ON.

### 4. Load the Extension
1. Click the "Load unpacked" (Загрузить распакованное расширение) button in the top-left corner.
2. Select the folder containing the extension files (the one with manifest.json inside).

That's it! The extension icon will appear in your list, and you're ready to convert.

## How to Use

1. Select any text on a webpage containing a price (e.g., 100$, 50 EUR, or just 2500).
2. Click the small floating currency icon that appears near your selection (or right-click and select "Convert currency").
3. Adjust the amount, swap currencies, or change the rate source directly in the pop-up box.
