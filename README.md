# Currency Quick Convert 🪙

A lightweight Google Chrome extension that allows you to instantly convert currencies by simply selecting text on any webpage. It features a smart overlay button for quick access and full support for multiple exchange rate sources.

## Features

- **Smart Detection:** Automatically recognizes numbers and currency symbols ($, €, ₽, £, ¥) in the selected text.
- **Quick-Access Button:** A small icon appears right next to your selection—no need to right-click and open the context menu.
- **Multiple Rate Sources:** Swap between Central Bank of Russia (CBR), Open ER API, and Currency-API on the fly.
- **Rate Pinning:** Save your favorite exchange rate provider as the default.
